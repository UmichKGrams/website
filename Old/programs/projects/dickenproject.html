Project: The Moon Scene  Date: 10/19/01
Project Head: Naomi Yodkovik

The mothers of the class were running a fall festival where the kids did fall-related activities.  These included foods and crafts.  For the last half hour, we played a Bingo game with some fall "terms".

Project: Fall Festival  Date: 11/16/01
Project Head: Naomi Yodkovik

First we had a short question-answer session of facts about the moon (characteristics, etc.).  Then we put various planets' moons on display.  Finally, the kids painted their own styrofoam ball moons, named them, and gave them their own set of characteristics.

Project: Reading Time  Date: 12/12/01
Project Head: Naomi Yodkovik

Reading Time with Mrs. Tilburt's 2nd graders was great! The college volunteers paired up with the kids to read some classic books such as Amelia Bedelia and Encyclopedia Brown during their silent reading time. Afterwards, we joined the kids on the playground for some old-fashioned recess fun!
