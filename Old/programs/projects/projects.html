Classroom Projects

Two line description
Currently, we are organizing projects with each elementary school classroom... These projects range from science experiments, tye-dying, gardening, playing instruments, cleaning up a school, and the list goes on! They provide essential time for the pen pals (and other college students) to interact and get to know each other better. To make projects run smoothly, volunteers are needed to be replacement pen pals (for anyone who couldn't make it), run the project so students can participate with the kids, and organize/plan the details beforehand.
